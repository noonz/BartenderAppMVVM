# Bartender MVVM App

## Overview
This application uses SwiftUI to display a list of drinks using MVVM architecture. It retrieves data from TheCocktailDB 
(https://www.thecocktaildb.com/api.php).  

## Dependencies  
**KingFisher:** Downloading/Displaying image urls
